# SCT_DS_3 - Decision Tree Classifier

## Objective
Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on demographic and behavioral data.

## Dataset Used
Bank Marketing Dataset (UCI Machine Learning Repository)

## Tools and Libraries Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Machine Learning Model
Decision Tree Classifier

## Project Workflow
1. Load the Bank Marketing dataset
2. Perform data preprocessing
3. Convert categorical data into numerical format
4. Split dataset into training and testing sets
5. Train the Decision Tree model
6. Predict customer purchase behavior
7. Evaluate model accuracy
8. Visualize the Decision Tree

## Outputs
- Accuracy Score
- Classification Report
- Decision Tree Visualization

## Conclusion
The Decision Tree Classifier was successfully built and trained using the Bank Marketing dataset to predict customer purchase behavior based on demographic and behavioral information.
